Question-1: What are the different ways to select an element in the DOM?


The different ways to select an element in the DOM are 1. getElementById(), 2. getElementsByClassName(), 3. getElementsByTagName(), 4. querySelector() 5. querySelectorAll() 6. getElementsByName()


Question-2: What is the difference between innerHTML, innerText, and textContent ?


innerHTML renders HTML tags as actual elements and innerText does not include HTML tags.


Question-3: What is event delegation in the DOM?


Event Delegation is a technique in JavaScript where you attach an event listener to a parent element instead of each child element.


Question-4: What is event bubbling in the DOM?


Event Bubbling is a process in the DOM where an event starts from the target element
